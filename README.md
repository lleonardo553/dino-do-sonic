## 🎮 Como Jogar

| Plataforma | Controle |
|------------|----------|
| **Desktop** | Pressione `ESPAÇO` ou `SETA PARA CIMA` para pular |
| **Mobile** | Toque na tela para pular |

### Objetivo
- Desvie dos obstáculos pulando no momento certo
- Sobreviva o máximo possível para acumular pontos
- Alcance novos níveis para desbloquear transformações do Sonic

## ⚡ Curva de Dificuldade

O jogo possui duas fases de dificuldade distintas:

### Fase 1: Início Suave (Nível 0 - 2.499)
- Velocidade inicial lenta e confortável
- Obstáculos espaçados e previsíveis
- Aceleração gradual para permitir adaptação

### Fase 2: Pico de Dificuldade (Nível 2.500+)
- 🔥 **Aumento significativo de velocidade**
- Obstáculos mais próximos entre si
- Maior variedade de tipos de obstáculos (voadores e pilares altos)
- Velocidade máxima limitada em 12 para manter jogabilidade

## 🎨 Tipos de Obstáculos

| Tipo | Aparência | Comportamento |
|------|-----------|---------------|
| **Espinho** | 🟤 Triângulo marrom | Obstáculo terrestre padrão |

| **Pilar Alto** | ⬛ Retângulo cinza | Obstáculo alto que exibe precisão |
